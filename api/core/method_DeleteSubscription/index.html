删除订阅信息

接口说明​

调用该接口删除订阅信息。

URI​

delete /subscriptions/{id}

请求参数​

名称参数位置类型描述是否必须

| id | path | string | 订阅id | Required |

| source | query | string | 来源id | false | | owner | query | string | 用户id | false |

响应​

响应<200>​

Code描述类型
200OKObject(v1DeleteSubscriptionResponse)

v1DeleteSubscriptionResponse​

字段名类型描述
idstring订阅id
statusstring状态
