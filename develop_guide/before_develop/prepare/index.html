插件开发​

插件开发部分介绍如何通过插件扩展机制来为平台提供扩展能力。

前提条件​

  • 已正确安装 git
  • 已正确安装 make
  • 已正确安装 go 1.16+
  • 已正确安装 protoc
  • 已正确安装 helm3
  • 已正确安装 docker
  • 已正确安装 tkeel 最新版
  • 可用的私有docker仓库或者docker hub账号（需本地登录docker hub）
